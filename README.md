Plany zajęć dla teleinfy na [GH Pages](https://matikd13.github.io/usos_scraper/).

## Jak używać

W urls.py wpisujemy rok (np. 25/26) i kierunki jakie chcemy. Zmienna PLANS zawiera przykład dla teleinfy. Potem odpalamy `uv run run_from_urls.py --urls urls.py --out dist` i gotowe.
